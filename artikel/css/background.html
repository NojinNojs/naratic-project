Mengubah Background Dengan CSS


CSS memungkinkan Anda untuk mengubah background elemen HTML dengan berbagai cara. Anda dapat mengubah warna, gambar, dan bahkan posisi background.

Contoh 1: Mengubah Warna Background

Berikut adalah contoh kode CSS untuk mengubah warna background: 

        .isi-artikel {
          background-color: #f0f8ff;
        }

Kode di atas akan mengubah warna background div menjadi biru muda. Anda dapat menggunakan kode warna hex, RGB, atau nama warna untuk menentukan warna background.

Contoh 2: Menambahkan Gambar Background

Berikut adalah contoh kode CSS untuk menambahkan gambar background: 

        .isi-artikel {
          background-image: url("images/background.jpg");
        }

Kode di atas akan menambahkan gambar "background.jpg" sebagai background div. Anda dapat menggunakan gambar dengan format JPG, PNG, atau GIF.

Contoh 3: Mengubah Posisi Background

Berikut adalah contoh kode CSS untuk mengubah posisi background: 

        .isi-artikel {
          background-position: center top;
        }

Kode di atas akan memposisikan gambar background di tengah atas div. Anda dapat menggunakan nilai "left", "right", "top", "bottom", "center", dan "px" untuk menentukan posisi background.

Kesimpulan

CSS menyediakan banyak cara untuk mengubah background elemen HTML. Anda dapat menggunakan berbagai properti CSS untuk mengubah warna, gambar, dan posisi background.

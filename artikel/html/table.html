Definisi Table HTML

Table HTML adalah elemen yang digunakan untuk membuat tabel data. Tabel HTML terdiri dari baris ( ) dan kolom ( ). Tabel HTML dapat digunakan untuk menampilkan data tabular seperti data statistik, daftar produk, dan lain-lain.

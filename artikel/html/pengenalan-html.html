Pengenalan HTML

HTML adalah singkatan dari HyperText Markup Language, yaitu bahasa pemrograman yang digunakan untuk membuat struktur dan isi halaman web.

HTML terdiri dari tag-tag yang digunakan untuk mendefinisikan berbagai elemen halaman web, seperti judul, paragraf, gambar, dan tautan.

Contoh Tag HTML

  • <h1> - Judul utama
  • <p> - Paragraf
  • <img> - Gambar
  • <a> - Tautan

Struktur Dasar Halaman HTML

Berikut adalah struktur dasar halaman HTML:

  • <html> - Tag pembuka HTML
  • <head> - Bagian kepala HTML, berisi informasi tentang halaman
  • <title> - Judul halaman
  • </head> - Tag penutup kepala HTML
  • <body> - Bagian isi HTML, berisi konten halaman
  • </body> - Tag penutup isi HTML
  • </html> - Tag penutup HTML
